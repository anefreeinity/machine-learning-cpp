# steps to follow.

1. extract mnist.zip in this directory.

2. you will get a folder called mnist, containing four files.
    i. train-images-idx3-ubyte.gz
    ii. train-labels-idx1-ubyte.gz
    iii. t10k-images-idx3-ubyte.gz
    iv. t10k-labels-idx1-ubyte.gz

3. extract all .gz files under mnist folder.

4. you will get four binary files.
    i. train-images-idx3-ubyte
    ii. train-labels-idx1-ubyte
    iii. t10k-images-idx3-ubyte
    iv. t10k-labels-idx1-ubyte

5. copy and past this four files in emnist-binary directory, make sure that names are matching if not then rename.

6. delete extra files and folders.
